# Contact Book MERN Project

Simple full-stack contact management app built with MERN stack.

## Features
- Add, edit, delete contacts
- Real-time updates (no page refresh needed)
- Works with MongoDB

## Tech Used
**Frontend**: React + Vite  
**Backend**: Node.js + Express + MongoDB + Mongoose  
**API**: Axios

## How to Run Locally

### Backend
cd backend
npm install
npm start

Runs on `http://localhost:5000`

### Frontend
cd client
npm install
npm run dev

Runs on `http://localhost:5173`
